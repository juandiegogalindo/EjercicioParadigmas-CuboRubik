# EjercicioParadigmas - CuboRubik

## 1. Nombre del Proyecto

**EjercicioParadigmas - CuboRubik**

Proyecto académico en **Java** que implementa un ejercicio relacionado con la teoría o simulación del **Cubo de Rubik**, utilizando los principios del paradigma de programación orientada a objetos y estructuras de datos.


## 2. Características

Este proyecto incluye:

- Implementación de la lógica asociada al **Cubo de Rubik** en **Java**.
- Organización del código en paquetes, clases y métodos orientados a objetos.
- Uso de estructuras de programación para representar el estado y rotaciones del cubo.
- Aplicación de control de flujo y manipulación de datos para simular movimientos del cubo.
- Código compatible con compilación y ejecución mediante IDE o herramientas de construcción como Ant. :contentReference[oaicite:0]{index=0}

## 5. Teoría del Juego en Base a la Programación

El Cubo de Rubik es un rompecabezas mecánico tridimensional compuesto por piezas que pueden girar alrededor de un núcleo central. La simulación de este cubo en un programa requiere representar cada cara, pieza y rotación mediante estructuras de datos y métodos que manipulan sus estados.

Conceptos Programáticos Fundamentales
1. Modelado de Estado:
- El cubo se representa internamente mediante estructuras que reflejan la orientación y posición de cada una de sus piezas (caras, aristas, esquinas).
2. Algoritmos de Rotación:
- Las rotaciones de caras o secciones del cubo se implementan como funciones que transforman el estado actual del cubo según reglas predefinidas.
3. Control de Flujo y Lógica:
- Para aplicar secuencias de movimientos o resolver partes del cubo, se utilizan bucles (for, while) y condicionales (if) que determinan qué pasos ejecutar y en qué orden.
4. Paradigma Orientado a Objetos:
- Se aprovecha la encapsulación y abstracción para definir clases que representan elementos del cubo (por ejemplo, Cubo, Cara, Movimiento), métodos para manipularlos y constructores para inicializar su estado.

Este ejercicio desarrolla competencias en estructuras de datos complejas, manipulación de objetos y aplicación de algoritmos que simulan procesos reales, vinculando un problema popular con fundamentos de programación avanzada.
